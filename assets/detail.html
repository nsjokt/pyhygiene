pyhygiene

여러 파이썬·흩어진 가상환경·수십 GB 캐시를 자동화는 안 깨고 안전하게 정리하는 Claude 스킬

무료 MIT 오픈소스 macOS · Linux 코딩 / 에이전트

이런 적, 있으시죠?

맥에서 파이썬 좀 쓰다 보면 누구나 겪는 상황입니다.

  • 🐍 pyenv · Homebrew · python.org · 시스템 파이썬이 뒤섞여 `python3`가 뭘 가리키는지 모르겠다
  • 📦 프로젝트마다 만든 가상환경(venv)이 여기저기 흩어져 디스크를 갉아먹는다
  • 💽 ~/.cache 의 huggingface·uv·pip 캐시가 어느새 수십 GB
  • 😰 지우고는 싶은데 — cron·자동매매 봇·스케줄러가 깨질까 봐 손을 못 댄다

pyhygiene가 4단계로 해결합니다

진단 → 계획 → 정리 → 재발 방지. 위험한 단계일수록 더 보수적으로.

audit 진단

인터프리터·venv·캐시·예약 작업을 전수 조사 (읽기 전용, 아무것도 바꾸지 않음)

plan 계획

위험도순으로 정리 후보를 랭킹하고 회수 가능한 디스크 용량을 알려줌

clean 정리

기본은 미리보기(dry-run). 실행 시 백업 먼저 만들고 안전한 것만 삭제

guard 예방

가상환경 밖 설치 차단 등 재발 방지 가드레일을 멱등 설치

왜 믿고 맡길 수 있나

"정리 스크립트"와 다른 점 — 안전성이 코드에 박혀 있습니다.

  • 🛡️ 자동화 인지cron·launchd가 실제로 실행하는 인터프리터·venv는 삭제 후보에서 자동 제외합니다. `uv run`을 쓰는 래퍼 스크립트까지 따라가 찾아냅니다.
  • 💾 백업 우선무엇이든 지우기 전에, 다시 설치할 수 있는 패키지 목록을 백업 매니페스트로 남깁니다.
  • 🔒 자동 sudo 안 함관리자 권한이 필요한 삭제(예: python.org 프레임워크)는 직접 실행하실 스크립트로 안내합니다. 대신 실행하지 않습니다.
  • 🚫 비밀번호 안 받음채팅에 붙여넣은 비밀번호는 어떤 명령에도 사용하지 않습니다.

받으시는 것

  • python-audit 스킬 (Claude Code)받아서 스킬 폴더에 넣으면 끝. Claude에게 "파이썬 환경 정리해줘"라고 말하면 자동으로 동작합니다.
  • CLI(pyhygiene) 번들 포함스킬 안에 CLI가 들어 있어 PyPI·별도 설치 없이 단독 작동합니다. 시스템 Python 3.9 이상이면 OK.
  • 오픈소스 전체 공개코드·테스트 전부 공개: github.com/nsjokt/pyhygiene (MIT)

사용법

Claude에게 말로 시키거나, 터미널에서 직접.

# 터미널에서 (설치 없이)
uvx pyhygiene audit      # 진단
uvx pyhygiene plan       # 정리 계획 + 회수 용량
uvx pyhygiene clean      # 미리보기 (기본 안전)
uvx pyhygiene clean --apply   # 백업 후 실제 정리

자주 묻는 질문

Q. 제 자동매매 봇이나 cron 작업이 깨지지 않나요?
cron·launchd를 교차검증해서 사용 중인 인터프리터·venv는 절대 삭제하지 않습니다. 8개의 launchd 작업이 도는 실제 봇 환경에서 안전을 검증했습니다.
Q. 잘못 지우면 되돌릴 수 있나요?
삭제 전에 재설치 가능한 패키지 목록을 백업 폴더에 남깁니다. 캐시·--user 패키지는 다시 받으면 그만입니다.
Q. 비밀번호가 필요한가요?
아니요. 관리자 권한이 필요한 작업은 직접 실행하실 스크립트로 안내만 하며, 비밀번호를 받거나 쓰지 않습니다.
Q. 정말 무료인가요?
네. MIT 라이선스 오픈소스이며 전체 코드가 공개돼 있습니다.
Q. 어떤 OS를 지원하나요?
macOS와 Linux입니다(인터프리터·캐시 경로 기준).
